# TP - Sistemas de Gestion de Base de Datos

Grupo 6 - Tema 2

La actividad consiste en desarrollar el diagrama ER y los esquemas de las relaciones resultantes del siguiente dominio de problema:

Se trata de una base de datos para administrar las camas y pacientes de un hospital. El hospital se encuentra organizado en pisos, cada piso tiene camas, un conjunto de enfermeras y médicos de piso, cada enfermera y cada médico tienen un horario y enfermos a cargo.

Cada paciente tiene uno o varios diagnósticos y una agenda de medicación y tratamiento a recibir, estos tienen un estado (suministrado, pendiente, cancelado) y un medico que lo prescribió, una fecha y hora de prescripición y de suministro.

Cada paciente una cama, una fecha de ingreso y de egreso.
